Esta página web está sendo desenvolvida como parte do bloco 3 do bootcamp Java Fullstack da Generation Brasil

📌 Sobre o bloco 3

Estou começando o bloco 3 do bootcamp Java Fullstack, e estou animado para mergulhar em novos aprendizados. Neste bloco, vou explorar e aprender sobre HTML, CSS, JavaScript e React. Esses são componentes essenciais para o desenvolvimento web moderno, e estou ansioso para aplicar esses conhecimentos na criação de projetos dinâmicos e responsivos.

📌 Sobre os blocos anteriores

Já estudei durante dois blocos no bootcamp Java Fullstack. No bloco 1, tive a oportunidade de me aprofundar em Java, enquanto no bloco 2 foquei em banco de dados relacionais e em Spring Boot, incluindo Spring Security e testes unitários. Esses blocos me proporcionaram uma base sólida e me prepararam bem para os desafios do próximo estágio do bootcamp.

Caso você tenha interesse, conheça um pouco mais sobre a Generation Brasil através de depoimentos de alunos que já passaram pelos bootcamps oferecidos.
